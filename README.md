## Dataset
Input data is from [OGC IndoorGML website](http://www.indoorgml.net/).


## Prerequisites
- Python 3.8+
- Plotly


## Getting Started

- To install the packages type in the console
```
$ pip install -r requirements.txt
```
- To run the program
```
$ python main.py
```
